# Dhruv-C

A collection of C programs written while learning and practicing core C programming concepts.

## What's Inside
This repo contains beginner-to-intermediate level C programs covering:
- Control structures (loops, conditionals)
- Arrays and strings
- Functions and recursion
- Pointers and memory basics
- Basic data structures

## How to Run
Each `.c` file can be compiled and run independently using GCC:

```bash
gcc filename.c -o output
./output
```

## Goal
This repo is a personal practice log for strengthening fundamentals in C before moving into more advanced topics like data structures and systems programming.

## Status
Actively updated as I work through new problems and concepts.
